TechStack: MySQL, PHP, CSS, HTML

1. Service Provider Profiles
Users can add details like name, phone, address, upload NID card for verification (provider).

2. Service Categories
Users can browse available service categories such as cleaning, cooking, babysitting, plumbing, electrician, price/hour, provider profile etc.

3. Favorite Providers
Customers can add providers as their favorite providers.

4. Booking System
Customers can request a service from the provider by selecting service type, date and time. Service Providers can accept/reject booking requests. Customers can view booking status.

5. Admin Dashboard
Admin can manage users, approve service providers, view reports of the system.

6. Ratings and Reviews
After completing a booking, customers can rate the provider and leave feedback.

7. Notifications System
Users receive notifications in their profile for booking confirmations, cancellations.

8. Service History
Customers can view their previous bookings and service records.

9. Payment
Customers can pay with virtual currency (online payment simulation), see payment status, provide transaction ID
